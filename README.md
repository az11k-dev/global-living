# GlobalLiving

**GlobalLiving** is a web platform that helps users explore countries and cities around the world.  
The project provides essential information about cost of living, daily expenses, accommodation prices, and general lifestyle insights. It is built to support **students**, **remote workers**, **job seekers**, and anyone planning to move abroad.
